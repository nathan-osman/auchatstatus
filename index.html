What is This?

Have you ever found the StackExchange Chat platform lacking in critical features like seeing how far other users have read, and a "this user is typing notification?" If so, this userscript is for you!

It adds these features to the AskUbuntu Chat Room (coming soon to all chat rooms), and completely revitalizes your chat experience!

Requirements

Using this script is easy! All you need is one of the below extensions, depending on your browser

  • Chrome, Opera, Safari: Tampermonkey
  • Firefox: Greasemonkey

How to Install

Installing the actual userscript is easy as 1-2-3:

  1. Make sure you have a Userscript manager installed.
  2. Click the Install button above.
  3. Click Install again!
